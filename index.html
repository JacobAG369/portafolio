Tu-Turismo

Desarrollo de un Pipeline de Datos de Ventas E2E

Análisis exploratorio y procesamiento de datos para plataforma de gestión turística. Encargado de la extracción y limpieza de datos (ETL) desde MongoDB utilizando Python (Pandas/NumPy). Responsable del diseño de KPIs y generación de dashboards interactivos enfocados en traducir métricas de interacción de usuarios y popularidad de destinos en insights estratégicos para la toma de decisiones.

  • Análisis integral de datos de interacción de usuarios y destinos turísticos para identificar patrones geográficos, estacionalidad y oportunidades de mejora en la oferta de la plataforma.
  • Extracción, limpieza y transformación de datos (ETL) desde bases de datos no relacionales
  • Diseño e implementación de métricas e indicadores clave (KPIs) para evaluar el rendimiento de la plataforma
  • Generación de reportes automáticos y dashboards visuales para el panel de administración, traduciendo datos técnicos en insights de negocio claros para facilitar la toma de decisiones estratégicas.

Tecnologías clave: HTML, CSS, JavaScript, arquitectura MVC.

MediBook

Proyecto orientado al desarrollo web, analisis de datos y uso de patrones de diseño.

SupportPandas - Sistema de Tickets

Orientado a desarrollo backend y analisis de datos.

SupportPandas es una aplicación web orientada a empresas para la gestión de tickets de soporte técnico. El objetivo del proyecto fue simular un entorno real de mesa de ayuda, desde la creación del ticket por parte del usuario hasta su seguimiento, actualización de estado y resolución por el área de soporte.

  • Registrar incidencias y asignarlas a responsables.
  • Dar seguimiento al historial de cada caso y mantener trazabilidad completa de cada solicitud.
  • Dashboards administrativos que permiten visualizar operaciones.

Tecnologías clave: Python (Django), Seaborn, pandas, PostgreSQL.
