MediBook

Proyecto académico · Gestión de citas médicas

Sistema pensado para administrar citas médicas tanto presenciales como en línea, con diferentes tipos de usuarios (pacientes, médicos y administración). El enfoque principal fue organizar el flujo completo de una cita, desde la solicitud hasta la confirmación.

  • Manejo de roles y permisos según el tipo de usuario.
  • Flujos separados para citas presenciales y en línea.
  • Aplicación de patrones de diseño a nivel de estructura del sistema.

Tecnologías clave: Java/Python, bases de datos, patrones de diseño.

Proyecto académico · Java
